## Parte 1: Quiz Teórico (20 perguntas)

**1. O que é uma Tabela Dinâmica?**
a) Uma tabela normal.
b) Uma ferramenta que resume dados em segundos.
c) Um gráfico.
d) Um antivírus.

**2. Para que serve uma Tabela Dinâmica?**
a) Criar documentos.
b) Agregar e resumir grandes volumes de dados.
c) Enviar e-mails.
d) Criar websites.

**3. O que é um Dashboard em Excel?**
a) Um gráfico único.
b) Um painel interativo com métricas e visuais conectados a dados.
c) Um antivírus.
d) Um tipo de planilha básica.

**4. O que faz a função SOMASE?**
a) Soma todos os valores.
b) Soma valores que atendem a um critério.
c) Conta células.
d) Calcula média.

**5. O que é uma Macro no Excel?**
a) Um tipo de gráfico.
b) Uma gravação de ações repetitivas para automatizar tarefas.
c) Um antivírus.
d) Uma fórmula.

**6. O que é VBA (Visual Basic for Applications)?**
a) Um antivírus.
b) A linguagem de programação para criar macros.
c) Um tipo deplanilha.
d) Um tipo de e-mail.

**7. O que é um "Slicer" em Tabelas Dinâmicas?**
a) Um tipo de faca.
b) Um filtro visual interativo.
c) Um antivírus.
d) Uma fórmula.

**8. Para que serve a Formatação Condicional?**
a) Formatar todo o documento.
b) Aplicar estilos automaticamente baseado em condições (ex: valores altos em vermelho).
c) Um antivírus.
d) Criar gráficos.

**9. O que é "Validação de Dados" no Excel?**
a) Verificar erros de ortografia.
b) Restringir o tipo de dado que pode ser digitado.
c) Um antivírus.
d) Criar macros.

**10. Qual função conta células que atendem a um critério?**
a) SOMA
b) CONT.SE
c) MÉDIA
d) PROCV

**11. O que é um gráfico dinâmico?**
a) Um gráfico que se move sozinho.
b) Um gráfico conectado a uma Tabela Dinâmica.
c) Um antivírus.
d) Uma Macro.

**12. O que faz o recurso "Agrupar" em Tabelas Dinâmicas?**
a) Agrupar células aleatoriamente.
b) Agrupar datas em meses/anos ou números em faixas.
c) Um antivírus.
d) Uma fórmula.

**13. O que é Power Query?**
a) Um antivírus.
b) Uma ferramenta para importar e transformar dados.
c) Um tipo de gráfico.
d) Uma Macro.

**14. O que é Power Pivot?**
a) Um antivírus.
b) Uma ferramenta para criar modelos de dados avançados.
c) Um tipo deplanilha.
d) Uma fórmula.

**15. O que é um "KPI" em dashboards?**
a) Um antivírus.
b) Uma métrica-chave de desempenho.
c) Um tipo de gráfico.
d) Uma fórmula.

**16. Para que serve a função PROCV?**
a) Buscar valores em uma tabela.
b) Criar gráficos.
c) Um antivírus.
d) Criar macros.

**17. O que é "Gráfico de Combinação"?**
a) Um gráfico misto (barras e linhas).
b) Um antivírus.
c) Um tipo deplanilha.
d) Uma fórmula.

**18. O que é uma "Tabela" no Excel (não dinâmica)?**
a) Um intervalo formatado como tabela estruturada.
b) Um antivírus.
c) Um gráfico.
d) Uma Macro.

**19. O que faz a função SE?**
a) Busca valores.
b) Cria condições lógicas (Se... Então...).
c) Soma valores.
d) Conta células.

**20. Para que serve o recurso "Segmentação de Dados"?**
a) Cortar dados.
b) Criar filtros visuais interativos para dashboards.
c) Um antivírus.
d) Criar macros.
